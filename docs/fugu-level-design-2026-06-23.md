## Sharp suggestions
- CURRICULUM SPINE (the whole point): make L2/L3/L4 each ADD one verb and RE-USE the prior level's idea, so the four screens read as one escalating lesson. L1 teaches 'read one rhythm.' L2 = same climb/rest rhythm but the rolling-skull lane becomes an ARROW lane (commit through impaired vision). L3 = 'the screen is a clock' (wait from L1 + commit-through-danger from L2, now with no enemies). L4 boss attacks each echo an earlier lesson: slash = L1 skull dodge, charge = L2 arrow-lane read, leap = L3 timer/lock-on read. This is the highest-leverage change - it converts four disconnected levels into a coherent difficulty curve.
- L2 FIX - make the FOG LADDER the main event, not decoration. Put one mandatory ladder through the fog band (Fugu: x176, y160->120) with a safe rest platform immediately before AND after it, so the decision is literally 'wait, read the archer, commit upward.' Keep exactly 2 archers (one horizontal-lane teacher BEFORE the fog at ~144f period/30f telegraph, one diagonal fog-tester offset +72f), give each a FIXED lane pattern, and guarantee at least one clean 64f climb window so the two arrows never both cover the fog ladder on loop 1. Turn the crate into bait (a knife/axe pickup that the archer's arrow can break) placed right before the climb, so the player SEES the danger lane before risking it.
- L3 FIX - run the whole level on ONE visible master clock with exactly 3 timed systems (lower pendulum 120f, crushing gear 240f, elevator 240f - elevator is traversal/non-damaging and shares the rhythm so it isn't a 4th idea). Compose it as four beats bottom-to-top: (1) isolated pendulum read with 2 free swings to watch, (2) elevator commitment that lines up with the crusher being open, (3) crusher as a FAIR GATE (always visible, 30f warning shake, knockback throws you back onto the approach ledge not down the whole level), (4) final exam = upper pendulum same 120f period but offset 60f so the player recognizes the machine but can't autopilot. No enemies - every death must read as 'I mistimed it.'
- L4 FIX - make the Hollow King a strict finite-state boss keyed to weak-points-remaining, NOT a random-attack soup, with no added platforms or hazards (arena stays flat). 4 alive = intro (slash+charge, 90f telegraphs); 3-2 alive = add leap, telegraphs shrink to 72f, max 2-move chains; 1 alive = add shield-bash as an anti-camping tool only, 60f telegraphs. Each weak point opens for a fixed window tied to a specific attack's recovery (slash->nearest sigil, charge->far sigil, leap->center sigil), and all-four-hit triggers a 180f / 2x-damage stagger. This forces pattern mastery instead of raw DPS and turns the fight into the curriculum's final exam.
- BUILD A DATA-DRIVEN FRAME TABLE and ship LOOP 2 as the real test of the design - before finishing all content. One row per hazard/attack with columns for telegraph/active/recovery/safe-window at loop 1/2/3, read by the game so loop scaling is data, not hardcoded guesses. Loops then re-contextualize the SAME screens via tighter windows + new offsets + pattern variants (red/wild rolling skulls on L1, an alternate high/low arrow lane on L2, crusher double-tap + rotating timing charts on L3, boss telegraph-shrink + changed opener/weak-point order on L4) - keeping a hard readability floor (never sub-12f reactions, min ~30f safe windows, HP scaling capped ~+10-15%/loop). Target feel: 'I understand this screen, but it's meaner.' Note: no full-loop playtest has ever run, so playtest the proposed geometry before committing it.
